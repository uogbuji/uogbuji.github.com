1 User-Contributed Notes and Documentation

4Suite users are encouraged to submit useful information related to 4Suite installation and usage by editing this page. This is not a forum to ask questions, though; use the 4Suite mailing list for that.

Eventually, the topics on this page can be better organized and moved to their own pages.

1.1 Installing PyGreSQL

If you want to use the Postgres driver for the 4Suite repository, you need to have the PyGreSQL (the PostgreSQL bindings for Python) installed. As of mid-2002, PyGreSQL installation from source can be tricky. There are 2 main problems:

  • Paths to lib and include directories are hard-coded in the setup.py file;
  • Postgres no longer installs postgres.h, which PyGreSQL needs.

Consequently, you may need to edit the PyGreSQL setup.py so that it has the correct paths. On my system, I happen to have the Postgres source in /usr/local/src, so this is what the relevant part of my setup.py looks like (actually lines 32-36 of the 3.2 version):

        include_dirs=['/usr/local/pgsql/include','/usr/local/src/postgresql-7.1.2/src/include']
        library_dirs=['/usr/local/pgsql/lib']
        optional_libs=['pq']
        data_files = []

1.2 Setting Up A Postgres User

Seeing errors like _pg.error: FATAL 1:  IDENT authentication failed for user "foo" when you try to run 4ss_manager?

Postgres has its own set of users and access permissions. It will pick up the current userid from the environment unless you have forced a different one with <User> in your 4ss config file. In either case, the user must exist in Postgres and must have permission to create databases. You can set up a new user in Postgres by running createuser, which should be in your pgsql/bin directory. You will need to connect as the postgres/pgsql user in order to run this the first time. 

% echo $USER 
foo 
% createuser --username=pgsql 
Enter name of user to add: foo 
Shall the new user be allowed to create databases? (y/n) y 
Shall the new user be allowed to create more new users? (y/n) y 
CREATE USER 
% echo "hooray! now I can use 4ss_manager!" 
hooray! now I can use 4ss_manager!

contributed by Mike Brown

1.3 Enabling Triclops on Cygwin

Here are instructions for enabling Triclops on a Cygwin installation of 4Suite.

  1. You will need to install the "jpeg: A library for manipulating JPEG image format files" package from Cygwin. This can be found under the "Graphics" category in the Cygwin setup.exe program.
  2. Download the latest graphviz binary for Cygwin from: http://www.bell-labs.com/cgi-user/wwexptools/gensnapshot?graphviz
  3. Unpack contents to a folder of your choice in the Cygwin directory tree, e.g. /usr/bin/graphviz
  4. In the 4Suite Dashboard, go to the "Configure" page and modify the "'dot' Executable directory" field to point to the location of dot.exe in your graphviz folder, e.g. /usr/bin/graphviz/bin. Also on this page is the "Rotate RDF Graphs 90 degrees?" option. In my experience, it looked better when set to "No". YMMV
  5. Wait a while, or just run "4ss_manager restart". At this point, Triclops will run, but the results will not be nice, unless you have the TrueType Times font installed in a place where dot.exe can find it. Also, an error message will be printed to the console from which you ran 4ss_manager, indicating that the font could not be found in the search path. The solution, then, is to create one of those directories and place the font in there.
  6. Create the following directory in your Cygwin directory tree: /usr/X11R6/lib/X11/fonts/TrueType
  7. Copy the times.ttf file into that folder. Since you're on Windows, you should already have this file. On my computer (WinXP), it was found in C:\Windows\Fonts
  8. Try viewing the Triclops page again, and things should look much better now.

contributed by Evan Lenz

1.4 Setting up Triclops on Debian

The Triclops install instructions in the UNIX install guide should all work. Use the following apt command to get Graphviz and the fonts it needs 

apt-get install graphviz msttcorefonts

contributed by Eric van der Vlist
